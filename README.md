# 💰 Islam and Economic Affairs — الإسلام والأوضاع الاقتصادية

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الإسلام والأوضاع الاقتصادية (Islam and Economic Affairs)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on Islamic economics: zakat, prohibition of riba, social justice, wealth distribution, work ethic, poverty elimination
- Economic wisdom section with problem/solution format
- Daily habit tracker, knowledge quiz, selected duas
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes, Narrator, Responsive + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
